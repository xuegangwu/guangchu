## 🔧 配置步骤

### 第 1 步：安装依赖

```bash
cd /home/admin/openclaw/workspace/projects/光储龙虾

# 安装翻译库
pip3 install googletrans==4.0.0-rc1
pip3 install deepl
pip3 install requests
pip3 install beautifulsoup4
```

### 第 2 步：配置翻译 API

```bash
# 复制配置模板
cp config/translation.json.example config/translation.json

# 编辑配置文件
nano config/translation.json
```

**配置示例**（使用 Google Translate）:
```json
{
  "google": {
    "enabled": true
  },
  "deepl": {
    "enabled": false,
    "api_key": ""
  },
  "baidu": {
    "enabled": false,
    "appid": "",
    "secret": ""
  },
  "youdao": {
    "enabled": false,
    "app_key": "",
    "secret": ""
  }
}
```

### 第 3 步：测试翻译服务

```bash
cd scripts
python3 translation_service.py
```

**预期输出**:
```
原文：太阳能电站投资评估系统
英文：Solar power station investment evaluation system
日文：太陽光発電所投資評価システム
```

## ⚠️ 注意事项

### 1. 翻译频率限制

- **Google Translate**: 免费但有频率限制
- **DeepL**: 50 万字符/月（免费）
- **百度**: 200 万字符/月（免费）
- **有道**: 100 万字符/月（免费）

**建议**: 使用缓存，避免重复翻译

### 2. 网站抓取

- 遵守 robots.txt
- 控制抓取频率
- 添加 User-Agent
- 处理反爬机制

### 3. 数据质量

- 检查翻译质量
- 过滤无效新闻
- 去重处理
- 验证链接有效性
