## 📱 多平台同步

### 生成流程

```
每天 18:00
    ↓
generate-visual-diary.py
    ↓
生成可视化日记 HTML
    ↓
保存到 diary/YYYY-MM-DD-visual.html
    ↓
更新 GitHub README
    ↓
生成公众号版本
    ↓
生成 Cloud Hub 版本
    ↓
完成！
```

### 导出格式

| 平台 | 格式 | 文件 |
|------|------|------|
| **GitHub** | HTML | `diary/YYYY-MM-DD-visual.html` |
| **公众号** | Markdown | `export/wechat/YYYY-MM-DD-wechat.md` |
| **Cloud Hub** | HTML | `export/wechat/YYYY-MM-DD-cloudhub.html` |
| **GitHub README** | Markdown | 自动更新顶部 |

## 🔧 技术实现

### 生成脚本
```python
scripts/generate-visual-diary.py
```

### 主要功能
1. **获取 Git 提交** - 自动收集当天代码提交
2. **获取完成任务** - 从任务列表提取
3. **获取重要进展** - 检测重要文件创建
4. **生成统计** - 计算提交数、文件变更等
5. **生成 HTML** - 创建福盛风格的可视化页面
6. **保存日记** - 保存到 diary 目录

### 自动运行
```bash
# 定时任务（每天 18:00）
0 18 * * * python3 scripts/generate-visual-diary.py
```
