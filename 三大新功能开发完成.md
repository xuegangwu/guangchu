## 2️⃣ 中文信息源

### 📖 功能说明

抓取中文媒体新闻，补充英文信息源的不足。

### 🚀 使用方法

```bash
cd /home/admin/openclaw/workspace/projects/光储龙虾
python3 scripts/fetch-chinese-news.py
```

### 📰 支持的信息源

| 信息源 | 类型 | URL |
|--------|------|-----|
| 北极星储能网 | 储能媒体 | storage.bjx.com.cn |
| 索比光伏网 | 光伏媒体 | solar.ofweek.com |

### ✨ 特性

- 🇨🇳 **中文内容**：补充英文媒体的不足
- 🔄 **自动分类**：自动识别区域和类型
- 🔗 **链接整合**：合并到现有 JSON 文件
- 📊 **去重机制**：避免重复抓取

### 🎯 自动分类逻辑

**区域识别**：
- "欧洲"、"欧盟"、"德国" → Europe
- "美国"、"加州" → US
- "中国"、"国家能源局" → China
- "日本" → Japan
- "东南亚"、"越南" → Southeast Asia

**类型识别**：
- "政策"、"补贴"、"规划" → 政策
- "项目"、"签约"、"中标" → 项目
- "产品"、"发布"、"价格" → 产品

### 📊 输出示例

```json
{
  "title": "中国首个城市地下钒流电池系统投运",
  "link": "https://storage.bjx.com.cn/news/123456.html",
  "published": "2026-03-12T10:00:00",
  "summary": "中国已部署首个 1.25 MW 城市地下钒流电池系统...",
  "source": "北极星储能网",
  "region": "China",
  "type": "项目"
}
```

### ⚠️ 注意事项

中文网站结构可能变化，如抓取失败需调整选择器。

## 3️⃣ 自动更新索引

### 📖 功能说明

一键自动执行：抓取新闻 → 构建索引 → 生成报告

### 🚀 使用方法

```bash
# 手动执行
cd /home/admin/openclaw/workspace/projects/光储龙虾
python3 scripts/auto-update.py

# 定时执行（每天 8:00）
crontab -e
# 添加：0 8 * * * cd /home/admin/openclaw/workspace/projects/光储龙虾 && python3 scripts/auto-update.py
```

### 🔄 执行流程

```
自动更新脚本
    ↓
1. 抓取英文新闻（PV Magazine, Energy Storage News）
    ↓
2. 抓取中文新闻（北极星储能网，索比光伏网）[可选]
    ↓
3. 构建搜索索引（SQLite FTS5）
    ↓
4. 输出统计信息
    ↓
完成！
```

### 📋 定时任务配置

已提供更新后的 crontab 配置文件：`crontab-updated.txt`

**安装方法**：
```bash
# 查看当前 crontab
crontab -l

# 编辑 crontab
crontab -e

# 粘贴 crontab-updated.txt 内容
```

**定时任务列表**：

| 时间 | 任务 | 说明 |
|------|------|------|
| 每天 8:00 | auto-update.py | 自动更新索引 |
| 每天 8:00 | fetch-news.py | 抓取英文新闻 |
| 每天 8:15 | fetch-chinese-news.py | 抓取中文新闻 |
| 每天 8:30 | build-index.py | 构建索引 |
| 每天 8:30 | generate-daily.py | 生成日报 |
| 每周一 9:00 | generate-weekly.py | 生成周报 |
| 每月 1 号 9:00 | generate-monthly.py | 生成月报 |

### 📝 日志管理

**日志位置**：`/tmp/guangchu-*.log`

**查看日志**：
```bash
# 查看最新日志
tail -f /tmp/guangchu-update.log
```

## ⚠️ 注意事项

### 1. 依赖安装

```bash
pip3 install flask requests beautifulsoup4
```

### 2. 端口占用

Web 服务默认使用 5000 端口，如被占用可修改：
```python
app.run(host='0.0.0.0', port=8080, debug=False)
```

### 3. 中文编码

确保终端支持 UTF-8：
```bash
export LANG=zh_CN.UTF-8
```

### 4. 网络访问

部分中文网站可能有反爬机制，建议：
- 降低抓取频率
- 添加 User-Agent
- 使用代理（如需要）
