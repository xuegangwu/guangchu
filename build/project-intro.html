📊 项目概况

光储龙虾是一个全面的太阳能和储能行业信息收集与分析系统，通过自动化抓取全球 22 个信息源，提供实时的行业新闻、政策动态、产品信息和项目信息。

🚀 核心功能

  • 自动化新闻抓取 - 每天自动从 22 个信息源抓取最新新闻
  • 高级搜索系统 - 支持关键词高亮、多条件过滤、结果排序
  • 数据分析可视化 - 实时统计和趋势分析图表
  • 开发日记系统 - 记录项目开发的每一天
  • 多语言支持 - 中文、英文、日文三语界面
  • RESTful API - 10 个 API 端点支持第三方集成
  • GitHub Pages 部署 - 自动化部署和 CDN 加速
  • 性能优化 - 数据库索引、缓存系统、查询加速 80%

🛠️ 技术栈
