📖 项目概述

光储龙虾是一个全球光伏 + 储能行业信息收集与分析系统，通过 GitHub Pages 实现静态网站部署。

技术栈

  • 前端: HTML5, CSS3, JavaScript (原生)
  • 后端: Python 3.9+
  • 数据库: SQLite FTS5
  • 部署: GitHub Pages
  • 版本控制: Git

✅ 最佳实践

HTML 结构

  • 使用语义化标签 (header, main, footer, section)
  • 保持合理的标题层级 (h1 → h2 → h3)
  • 添加适当的 meta 标签 (charset, viewport)

CSS 样式

  • 使用 CSS 变量管理主题色
  • 采用 Flexbox/Grid 布局
  • 实现响应式设计 (mobile-first)
  • 添加过渡动画提升体验

JavaScript

  • 避免全局变量污染
  • 使用事件委托优化性能
  • 添加错误处理
  • 本地存储用户偏好
💡 提示: GitHub Pages 是静态托管，不能运行服务器端代码。所有动态功能需通过 JavaScript 实现。

🔧 常见问题

1. 404 错误

问题: 页面部署后访问 404

解决方案:

  1. 检查文件是否在 gh-pages 分支
  2. 确认文件名大小写正确
  3. 等待 1-2 分钟 CDN 刷新
  4. 检查 GitHub Actions 部署日志

2. CSS/JS 不加载

问题: 页面样式或脚本未加载

解决方案:

  • 检查文件路径是否正确 (使用相对路径)
  • 清除浏览器缓存
  • 检查 MIME 类型设置

3. 中文乱码

问题: 中文显示为乱码

解决方案:

  • 确保文件保存为 UTF-8 编码
  • 添加 <meta charset="UTF-8">

🚀 部署指南

手动部署

# 1. 切换到 gh-pages 分支
git checkout gh-pages

# 2. 复制构建文件
cp build/* .

# 3. 提交更改
git add .
git commit -m "🚀 部署新版本"

# 4. 推送到远程
git push origin gh-pages

自动部署 (GitHub Actions)

name: Deploy to GitHub Pages
on:
  push:
    branches: [main]
jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Deploy
        uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./build
