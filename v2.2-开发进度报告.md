### 核心功能

#### 1. 高级搜索类
```python
class AdvancedSearch:
    - highlight_keywords()      # 关键词高亮
    - search()                  # 高级搜索
    - get_search_history()      # 获取历史
    - get_search_suggestions()  # 搜索建议
    - search_stats()            # 搜索统计
```

#### 2. Web 搜索界面

- 结果列表展示
- 统计面板

## 🔍 搜索功能演示

### 基础搜索
```python
search = AdvancedSearch()
results = search.search("solar", limit=20)
```

### 带过滤的搜索
```python
results = search.search(
    "energy",
    filters={
        'date_from': '2026-03-01',
        'date_to': '2026-03-31',
        'region': 'US',
        'type': 'policy'
    },
    sort_by='date',
    limit=20
)
```

### 关键词高亮
```python
text = "Solar energy is important"
highlighted = search.highlight_keywords(
    text, 
    ["solar", "energy"]
)
# 结果："Solar energy is important" (带<mark>标签)
```
