## 🔍 诊断步骤

### 如果页面显示不正常，请执行：

**1. 强制刷新浏览器**
```
Windows/Linux: Ctrl + Shift + R
Mac: Cmd + Shift + R
```

**2. 清除浏览器缓存**
- Chrome: 设置 → 隐私和安全 → 清除浏览数据
- Firefox: 选项 → 隐私与安全 → 清除数据
- Safari: 偏好设置 → 隐私 → 管理网站数据

**3. 使用隐私模式**
- Chrome: Ctrl+Shift+N
- Firefox: Ctrl+Shift+P
- Safari: Cmd+Shift+N

**4. 验证 raw 访问**
```
https://raw.githubusercontent.com/xuegangwu/guangchu/gh-pages/diary/2026-03-14.html
```

如果 raw 访问正常但 GitHub Pages 不正常，说明是 CDN 缓存问题。

**5. 等待 CDN 更新**
- CDN 更新需要 5-10 分钟
- 预计生效时间：21:45-21:50
