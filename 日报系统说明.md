## 🔧 配置选项

### 修改生成时间

编辑定时任务：
```bash
crontab -e
```

修改时间（示例：改为 17:30）：
```bash
30 17 * * * cd /home/admin/openclaw/workspace/projects/光储龙虾 && python3 scripts/generate-daily-report.py
```

### 修改报告位置

编辑脚本中的配置：
```python
REPORTS_DIR = Path("/your/custom/path/reports/daily")
```

### 添加飞书推送

在脚本中实现 `send_to_feishu` 方法：
```python
def send_to_feishu(self, report: str):
    # 调用飞书 API 发送报告
    import requests
    
    webhook_url = "YOUR_FEISHU_WEBHOOK_URL"
    
    data = {
        "msg_type": "text",
        "content": {
            "text": report[:1000]  # 发送前 1000 字
        }
    }
    
    requests.post(webhook_url, json=data)
```
