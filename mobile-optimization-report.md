## 响应式断点

### 标准断点
- **Desktop**: > 768px
- **Tablet**: ≤ 768px
- **Mobile**: ≤ 480px
- **Small Mobile**: ≤ 375px (iPhone SE 等)

### 优化策略
1. **字体大小** - 随屏幕缩小而递减
2. **间距调整** - padding/margin 按比例缩小
3. **布局变化** - 多列 → 单列 / 水平 → 垂直
4. **网格系统** - 自适应列数 (4→2→1)
5. **导航优化** - 简化链接 / 垂直排列
