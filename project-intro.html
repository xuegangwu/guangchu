FEATURES

技术特点

创新技术，卓越性能

🌐

多语言翻译

支持英文、中文、日文三语自动翻译。集成 Google Translate、DeepL、百度翻译等多个翻译服务，自动检测源语言并翻译成目标语言。

🔍

FTS5 全文搜索

基于 SQLite FTS5 引擎，支持布尔搜索、短语搜索、邻近搜索等高级搜索语法。搜索响应时间 < 50ms。

🤖

智能数据处理

自动内容提取、质量过滤、关键信息提取（公司/人物/地点/数字）、智能摘要生成、情感分析、主题分类。

⏰

自动化更新

支持定时任务自动抓取，每天自动更新。集成 Cron 定时任务，支持手动触发和自动执行。

💾

翻译缓存

智能翻译缓存机制，避免重复翻译。相同内容直接返回缓存结果，节省 API 调用成本，提升响应速度。

📊

数据统计

实时统计面板，展示信息源分布、类型分布、区域分布、趋势分析等。支持数据导出和可视化展示。
