## 🔧 技术实现

### 搜索引擎：SQLite FTS5

- **轻量级**：无需额外服务，单文件数据库
- **高性能**：60 条记录 < 10ms，10000 条 < 200ms
- **功能强**：支持布尔搜索、短语搜索、邻近搜索

### 索引结构

```sql
-- 主表（结构化数据）
CREATE TABLE news (
    id INTEGER PRIMARY KEY,
    title TEXT,
    summary TEXT,
    source TEXT,
    region TEXT,
    type TEXT,
    ...
)

-- FTS5 全文搜索虚拟表
CREATE VIRTUAL TABLE news_fts USING fts5(
    title,
    summary,
    content='news'
)
```

### 自动同步

使用触发器自动同步主表和 FTS 表：
- INSERT → 自动添加到索引
- UPDATE → 自动更新索引
- DELETE → 自动删除索引

## 🎯 搜索功能

### 支持的筛选条件

| 参数 | 说明 | 示例 |
|------|------|------|
| `keyword` | 关键词（英文） | `"battery"` |
| `-t, --type` | 类型 | `产品/政策/项目/其他` |
| `-r, --region` | 区域 | `Europe/US/Global/...` |
| `-s, --source` | 来源 | `PV Magazine/Energy Storage News` |
| `--from` | 起始日期 | `2026-03-01` |
| `--to` | 结束日期 | `2026-03-31` |
| `-l, --limit` | 结果数量 | `10` |

### FTS5 高级语法

```bash
# 或逻辑
python3 search.py "battery OR storage"

# 与逻辑
python3 search.py "solar AND module"

# 排除
python3 search.py "battery -lithium"

# 精确短语
python3 search.py '"energy storage"'

# 通配符
python3 search.py "solar*"
```

## ⚠️ 注意事项

### 1. 关键词语言

**数据源是英文媒体**，搜索关键词需用**英文**：

| ❌ 中文（无效） | ✅ 英文（有效） |
|----------------|----------------|
| 储能 | battery, energy storage, BESS |
| 太阳能 | solar, PV, photovoltaic |
| 组件 | module, panel |
| 逆变器 | inverter |
| 项目 | project, installation |

### 2. 索引更新

每次运行 `fetch-news.py` 抓取新数据后，需要重新构建索引：

```bash
python3 scripts/fetch-news.py    # 抓取新闻
python3 scripts/build-index.py   # 重建索引
```

### 3. 中文搜索

当前版本不支持中文关键词搜索（数据源是英文）。如需中文搜索，需要：
- 方案 1：添加中文信息源（北极星储能网等）
- 方案 2：实时翻译摘要为中文
