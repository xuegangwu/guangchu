# 光储龙虾 - 全球光储信息追踪

## 任务目标
收集欧美日东南亚地区的光伏+储能（光储）政策、产品、项目信息，定期产出报告并发布到头条。

## 产出节奏
- **每日短报**：当日重要动态（500-1000字）
- **每周报告**：深度分析 + 趋势（3000-5000字）
- **每月报告**：综合综述 + 市场洞察（10000+ 字）

## 覆盖区域
- 🇪🇺 欧洲（EU）
- 🇺🇸 美国（US）
- 🇯🇵 日本（JP）
- 🌏 东南亚（SEA）

## 信息类别
1. **政策信息**：补贴政策、并网规定、碳关税、贸易壁垒等
2. **产品信息**：组件/逆变器/储能系统价格、新品发布、技术进展
3. **项目信息**：大型项目签约、EPC 中标、并网数据等

## 目录结构
```
光储龙虾/
├── sources/          # 信息源配置
├── reports/
│   ├── daily/        # 每日短报
│   ├── weekly/       # 每周报告
│   └── monthly/      # 每月报告
├── templates/        # 报告模板
└── README.md
```
