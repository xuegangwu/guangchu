## 🔧 配置选项

### 修改生成时间

编辑定时任务：
```bash
crontab -e
```

修改时间（示例：改为 17:30）：
```bash
30 17 * * * cd /home/admin/openclaw/workspace/projects/光储龙虾 && python3 scripts/generate-project-diary.py
```

### 修改条目数量

编辑脚本中的配置：
```python
# 修改最大条目数
entries = self.generate_diary_entries(max_entries=5)  # 改为其他数字
```

### 添加自定义内容

在 `generate_diary_entries` 方法中添加：
```python
entries.append({
    'type': 'custom',
    'emoji': '🌟',
    'title': '自定义内容',
    'time': '今天',
    'category': '自定义分类'
})
```
