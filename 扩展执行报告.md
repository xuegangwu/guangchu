```bash
cd /home/admin/openclaw/workspace/projects/guangchu/scripts

```

### 自动更新（包含所有信息源）

```bash
cd /home/admin/openclaw/workspace/projects/光储龙虾
python3 scripts/auto-update.py
```

### 多语言搜索

```bash
# 搜索英文
python3 scripts/search.py "solar policy"

# 搜索中文
python3 scripts/search.py "太阳能政策"

# 搜索日文
python3 scripts/search.py "太陽光政策"

# 按类型筛选
python3 scripts/search.py -t 政策

# 按区域筛选
python3 scripts/search.py -r China
```

### Web 搜索

```bash
# 启动 Web 搜索服务
python3 scripts/web-search.py

# 访问：http://localhost:5000
```

### 可选配置

如需使用其他翻译服务，编辑 `config/translation.json`:

```json
{
  "google": {
    "enabled": true
  },
  "deepl": {
    "enabled": false,
    "api_key": "YOUR_API_KEY"
  },
  "baidu": {
    "enabled": false,
    "appid": "YOUR_APPID",
```
