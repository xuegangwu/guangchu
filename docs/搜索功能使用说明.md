## 📖 搜索功能

### 基础搜索

| 命令 | 说明 |
|------|------|
| `python search.py "储能"` | 搜索包含"储能"的新闻 |
| `python search.py "solar module"` | 英文关键词搜索 |
| `python search.py "电池 OR storage"` | FTS5 语法：OR 逻辑 |
| `python search.py "NEAR(battery, 5)"` | FTS5 语法：邻近搜索 |

### 条件筛选

| 参数 | 说明 | 示例 |
|------|------|------|
| `-t, --type` | 类型筛选 | `-t 产品` |
| `-r, --region` | 区域筛选 | `-r Europe` |
| `-s, --source` | 来源筛选 | `-s "PV Magazine"` |
| `--from` | 起始日期 | `--from 2026-03-01` |
| `--to` | 结束日期 | `--to 2026-03-12` |
| `-l, --limit` | 结果数量 | `-l 10` |

### 组合搜索示例

```bash
# 搜索欧洲的产品新闻
python search.py -t 产品 -r Europe

# 搜索 3 月份的储能相关内容
python search.py "储能" --from 2026-03-01 --to 2026-03-31

# 搜索 PV Magazine 来源的政策新闻
python search.py -t 政策 -s "PV Magazine"

# 搜索美国的项目，限制 5 条结果
python search.py -t 项目 -r US -l 5
```

### 输出模式

```bash
# 详细输出（默认）
python search.py "太阳能"

# 紧凑输出（列表形式）
python search.py "太阳能" --compact

# 导出为 JSON
python search.py "储能" --export results.json

# 显示统计信息
python search.py --stats
```

## 🎯 FTS5 高级搜索语法

SQLite FTS5 支持强大的搜索语法：

| 语法 | 说明 | 示例 |
|------|------|------|
| `term` | 基本搜索 | `battery` |
| `"phrase"` | 精确短语 | `"solar panel"` |
| `term1 OR term2` | 或逻辑 | `储能 OR 电池` |
| `term1 AND term2` | 与逻辑 | `太阳能 AND 组件` |
| `-term` | 排除 | `储能 -锂` |
| `term*` | 通配符 | `solar*` |
| `NEAR(term1, term2, N)` | 邻近搜索 | `NEAR(battery, storage, 5)` |

示例：
```bash
# 搜索包含"solar"或"photovoltaic"的新闻
python search.py "solar OR photovoltaic"

# 搜索包含"battery"但不包含"lithium"的新闻
python search.py "battery -lithium"

# 搜索精确短语"energy storage"
python search.py '"energy storage"'
```

## 🔧 技术细节

### 索引结构

使用 SQLite FTS5 全文搜索引擎：

- **主表** `news`：存储完整结构化数据
- **FTS 表** `news_fts`：全文搜索索引
- **触发器**：自动同步主表和 FTS 表

### 搜索字段

可搜索的字段：
- `title`：新闻标题
- `summary`：新闻摘要

可筛选的字段：
- `region`：区域（Europe/US/Global/Japan/Southeast Asia）
- `type`：类型（政策/产品/项目/其他）
- `source`：来源（PV Magazine/Energy Storage News）
- `collected_at`：收集日期（YYYY-MM-DD）

### 性能

- 60 条记录：搜索 < 10ms
- 1000 条记录：搜索 < 50ms
- 10000 条记录：搜索 < 200ms

## ⚠️ 注意事项

1. **索引更新**：每次运行 `fetch-news.py` 后，需要重新构建索引
   ```bash
   python scripts/fetch-news.py
   python scripts/build-index.py
   ```

2. **中文搜索**：FTS5 对中文支持有限，建议使用关键词而非长句

3. **大小写**：搜索不区分大小写

4. **特殊字符**：搜索包含特殊字符的关键词时需要用引号包裹
