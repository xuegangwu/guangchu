# 光储龙虾 - 第三方平台 API 集成研究

## 1. 今日头条 (Toutiao)

### 现状
- 已完成：每日自动生成图文文章 ()
- 需完成：自动发布到今日头条

### API 方案

#### 方案 A: 开放平台 OAuth (推荐)
- 官网：https://open.toutiao.com/
- 需要企业认证
- 支持文章自动发布
- 申请流程：
  1. 注册开发者账号
  2. 创建应用 → 选择头条号
  3. 申请权限：文章发布
  4. 获取 AppID 和 AppSecret
  5. OAuth2.0 授权获取 access_token

#### 方案 B: 第三方平台 (如微微号)
- 使用第三方服务如 微盛、微微号
- 可能需要付费
- 功能有限

#### 方案 C: 草稿箱 + 手动发布
- 每天自动生成文章到草稿箱
- 手动复制发布
- 适合测试阶段
